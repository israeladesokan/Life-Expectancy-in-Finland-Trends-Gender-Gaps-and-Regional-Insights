## Project Overview
This Power BI dashboard explores **life expectancy trends in Finland**, highlighting how longevity varies by **gender**, **region**, and **time**.  
It also includes **forecasts to 2070** and supporting analyses such as **top causes of death** and **seasonal death occurrence**.

The goal of this project is to understand what drives longevity in Finland and communicate those insights through clear, interactive visuals.
